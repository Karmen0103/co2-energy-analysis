# CO₂ Emissions and Renewable Energy Analysis (2000–2022)

This project explores global CO₂ emissions and renewable energy cost trends using data from *Our World in Data*. The goal is to identify key drivers of emissions, evaluate country-level progress in reducing emissions, and analyse future cost trends of renewable energy technologies.

---

## 📊 Research Questions

1. What is the biggest predictor of a large CO₂ output of a country?  
2. Which countries are making the biggest strides in decreasing CO₂ output?  
3. Which non-fossil fuel energy technology will have the best price in the future?  

---

## 📁 Project Structure

- `CO₂__and_renewable_energy_analysis.ipynb` → Main Jupyter Notebook with full data analysis

---

## 📊 Data Sources

All datasets are from **Our World in Data**:

- CO₂ emissions and GDP data  
- Primary energy consumption data  
- Daily calorie availability data  
- Levelized cost of energy (LCOE) data  

---

## 🛠️ Methods Used

- Data cleaning and preprocessing (Pandas)
- Data merging and filtering
- Pearson correlation analysis
- Percentage change calculations
- Linear regression (NumPy polyfit)
- Log transformation for stable forecasting
- Data visualization (Matplotlib)

---

## 📈 Key Findings

- **Energy consumption per capita** is the strongest predictor of CO₂ emissions.
- Only a limited number of countries achieved significant reductions in emissions between 2000 and 2022.
- **Solar photovoltaic energy** shows the strongest long-term cost decline and highest potential for future competitiveness.
