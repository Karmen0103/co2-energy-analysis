# 📊 **CO₂ Emissions and Renewable Energy Analysis (2000–2022)**

## 📌 **Overview**
This project analyzes global CO₂ emissions and renewable energy cost trends using real-world datasets from Our World in Data. The goal is to identify key drivers of emissions, evaluate country-level changes, and explore long-term renewable energy cost trends.

## 📊 **Research Focus**
* Key drivers of CO₂ emissions  
* Countries with the largest emission reductions  
* Long-term cost trends of renewable energy technologies  

## 📁 **Data Sources**
* Our World in Data (CO₂ emissions, GDP, energy consumption, LCOE datasets)

## 🛠️ **Methods**
* Data cleaning and preprocessing (Python, Pandas)  
* Correlation analysis (Pearson)  
* Percentage change calculations  
* Linear regression modeling (NumPy)  
* Log transformation for stable forecasting  
* Data visualization (Matplotlib)  

## 📈 **Key Findings**
* Energy consumption per capita is the strongest driver of CO₂ emissions.  
* Only a limited number of countries achieved significant emission reductions.  
* Solar photovoltaic shows the strongest long-term cost decline among renewable technologies.  

## 📊 **Visualization**
* Basic visualizations including bar charts and a time-series line chart to support the analysis and illustrate key trends.

## 🎯 **Outcome**
This project demonstrates applied data analysis skills using real-world environmental and energy datasets, including the use of visualization to support insights.

## 📌 **Notes**
This project was completed as part of a data analysis course focusing on applied statistical methods and real-world datasets. The analysis reflects practical use of Python for data exploration, modeling, and visualization.- Only a limited number of countries achieved significant reductions in emissions between 2000 and 2022.
- **Solar photovoltaic energy** shows the strongest long-term cost decline and highest potential for future competitiveness.
